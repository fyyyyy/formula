```console
$ git clone https://github.com/tsoding/formula
$ cd formula
$ iexplore.exe ./index.html
```

## Convert obj to js models

Run obj2js.js with node to convert an .obj into .js format

```console
node ./obj2js.js ./PalmTree_1.obj
```
